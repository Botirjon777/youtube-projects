Click “Block” and watch the left box choke while transform glides.

Why?

  • left changes the page layout, so the browser must recalculate positions, repaint, then composite — all on the main thread, every single frame.
  • transform (and opacity) skip layout & paint. They run on the compositor thread, so they keep animating even when JavaScript jams the main thread.
  • Rule of thumb: animate only transform and opacity. Add will-change: transform to promote the layer.
